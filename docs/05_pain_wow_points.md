# Step 05. PainPoint / WowPoint

> 목적: `docs/04` 기존 여정에서 Pain·Wow를 추출하고, 우리 서비스의 개선·차별 방향으로 정리한다.

> **Guide:** 기존 여정에서 사용자가 **불편했던(Pain)** 점은 우리 서비스에서 개선하고, **좋았던(Wow)** 점은 가져와 매력 있고 가치 있는 서비스를 만든다. `docs/04`의 모든 여정·단계를 검토해 Pain/Wow를 빠짐없이 나열한다.

**프롬프트 예시**

```
docs/04_existing_customer_journey.md를 읽고 docs/05_pain_wow_points.md를 채워줘.
각 여정·단계에서 사용자가 불편했던 Pain, 좋았던 Wow를 모두 추출해 나열해줘.
하나로 합치거나 우선순위를 정하지 말고, 발견한 항목을 빠짐없이 bullet로 정리해줘.
구조와 상단 Guide는 유지하고, 본문만 채워줘.
```

## PainPoint

### 여정 1 — YouTube·검색 튜토리얼

- (여정 1, 1) 작곡하고 싶은데 **뭐부터 해야 할지 막막**하다
- (여정 1, 4) **기호·용어(Cmaj7, I-IV-V-vi, 전조, 세븐스 등)가 너무 많다**
- (여정 1, 4) 이해하려고 **멈추고 되감기를 반복**해야 한다
- (여정 1, 4) **용어 사전**이 따로 필요한 것 같다
- (여정 1, 5) 영상 속 코드를 **내가 직접 확인·들어볼 수 없다**
- (여정 1, 5) **보컬만 하는 사람**에게는 악기 중심 설명과 **거리가 있다**
- (여정 1, 6) **이론부터 전부 배워야 하나** 싶다
- (여정 1, 6) **당장 곡이 만들어지지 않는다** — 좌절
- (여정 1, 6) 영상 2~3개 시청 후 **피로**하거나, 북마크만 하고 **끝낸다**
- (여정 1, 7) 작곡을 **미루고** 커버곡·"아직 준비 중"으로 **실행이 정체**된다

### 여정 2 — 밴드·지인에게 물어보기

- (여정 2, 1) 밴드에 작곡하는 사람이 없으면 **내가 해야 한다**는 **부담**
- (여정 2, 3) 원하는 분위기를 **말로 설명하기 어렵다**
- (여정 2, 4) 코드표 **글자는 알겠는데 소리가 상상되지 않는다**
- (여정 2, 5) 들은 뒤에도 **"이게 맞나?"** — 본인이 **선택·수정할 수 없다**
- (여정 2, 6) 수정을 **계속 부탁하기 미안**하다
- (여정 2, 6) 결과물이 **"내 곡이 아닌 것 같다"**
- (여정 2, 6) 친구 **일정·기분**에 따라 반응이 달라진다
- (여정 2, 7) 친구가 바쁘면 **무기한 보류**
- (여정 2, 7) **"작곡은 친구 있는 사람만 하는구나"** — 소외감

### 여정 3 — DAW·작곡 앱

- (여정 3, 2) 첫 실행 시 **화면·메뉴가 많아** 압도된다
- (여정 3, 2) **트랙, MIDI, BPM** 등 용어를 모른다
- (여정 3, 3) 루프·샘플로는 만들기 쉬우나 **내 멜로디·내 코드가 아니다**
- (여정 3, 4) **어디를 눌러야 코드가 되는지** 모르겠다
- (여정 3, 4) 무작위 건반 클릭 후 **포기**
- (여정 3, 5) 앱 사용에 **이론 학습(여정 1)까지** 필요하다
- (여정 3, 6) 앱은 깔려 있지만 **열지 않거나** 루프곡만 만든다
- (여정 3, 6) **"작곡 앱 = 비트 만들기"** 인식이 고착된다
- (여정 3, 7) **자작 코드 진행 없이** 끝나거나 템플릿 수준에 그친다
- (여정 3, 7) **"진짜 작곡은 다른 레벨"** — 자신감 하락

### 여정 4 — 화성학·작곡 교재·강의

- (여정 4, 2) **공짜만 찾다가** 강의·교재 선택에서 포기
- (여정 4, 3) 머리로는 이해되는데 **귀와 손이 따라오지 않는다**
- (여정 4, 4) 과제는 맞는데 **듣기·청음**이 약하다
- (여정 4, 5) 밴드 일정(**"이번 주 데모"**)과 **이론 학습 속도**가 맞지 않는다 — "아직 3강인데 곡은 언제?"
- (여정 4, 6) 강의 **중도 이탈**, 책 **1/3만** 읽고 중단
- (여정 4, 6) **"작곡은 공부 끝난 사람만 하는 것"**처럼 느껴진다
- (여정 4, 7) 이론 조각만 남고 **첫 자작곡은 미완**
- (여정 4, 7) **"언젠가 다 배우면…"** — 장기 연기

### 여정 5 — 기존 곡 코드 베끼·변형

- (여정 5, 2) **탭·코드표를 처음** 본다 — 읽는 법을 모른다
- (여정 5, 2) **Capo** 등 용어를 모른다
- (여정 5, 3) 진행 **이름(코드 기호)** 은 모르겠다
- (여정 5, 4) 코드 하나 바꾸면 **어색한데 왜 그런지 설명 못 한다**
- (여정 5, 5) 멜로디·가사는 새로 써도 **화성은 원곡에 의존**
- (여정 5, 5) **"자작곡"이라 부르기 애매**하다
- (여정 5, 6) **표절 걱정**으로 중단
- (여정 5, 6) 베끼기 말고 **직접 고르고 싶은데** 방법이 없다

### 여정 6 — AI 챗봇·생성 AI

- (여정 6, 3) 결과가 **텍스트(코드표)뿐** — **소리가 없다**
- (여정 6, 4) AI가 준 진행이 **맞는지 내 귀로 판단하기 어렵다**
- (여정 6, 4) 확인하려면 **피아노 앱·친구** 등 **다른 수단**이 또 필요하다
- (여정 6, 5) **대화로 한 진행을 다듬기** 번거롭다
- (여정 6, 5) 재프롬프트 시 **맥락 유지가 불안정**하다
- (여정 6, 6) **텍스트 코드표만** 남거나 만족 못 하고 **다른 여정으로 회귀**

### 여정 7 — 포기·미루기

- (여정 7, 2) 1~2회 시도 후 **"생각보다 어렵다"**
- (여정 7, 3) **"작곡은 ○○(친구) 재능"** — **내 영역이 아니다** 자기귀인
- (여정 7, 4) 밴드에서 **"작곡 못 하는 사람"** 역할에 고정
- (여정 7, 4) 작곡 이야기를 **회피**
- (여정 7, 5) 미루는 이유가 **시간 부족이 아니라 진입장벽**
- (여정 7, 5) 방학·새 학기마다 재시도해도 **같은 패턴** 반복
- (여정 7, 6) **자작곡 0곡** — **시작조차 못 했다**
- (여정 7, 6) **첫 완성 경험** 목표 **미달성**

## WowPoint

### 여정 1 — YouTube·검색 튜토리얼

- (여정 1, 1) **작곡하고 싶다** — 멜로디·가사 아이디어가 떠오른다 (동기·기대)
- (여정 1, 2) **무료로 배울 수 있겠다** — 희망
- (여정 1, 3) 조회수 많은 입문 영상이면 **"이거면 따라 할 수 있겠지"**

### 여정 2 — 밴드·지인에게 물어보기

- (여정 2, 2) 친구에게 맡기면 될 것 같다 — **안도**
- (여정 2, 5) 친구가 기타로 쳐 주거나 합주할 때 **"오, 이 느낌!"**
- (여정 2, 7) 친구 손에 의존해도 **"일단 한 곡은 했다"**

### 여정 3 — DAW·작곡 앱

- (여정 3, 1) **"앱 하나면 되겠지"** — 작곡이 가까워 보인다
- (여정 3, 3) 루프·프리셋으로 **만들기는 쉽다** — 즉시 뭔가 나온다

### 여정 4 — 화성학·작곡 교재·강의

- (여정 4, 1) **"제대로 배우면 평생 쓸 수 있겠다"** — 체계 학습에 대한 기대
- (여정 4, 2) **"시간·비용을 들여도 투자할 가치 있다"**

### 여정 5 — 기존 곡 코드 베끼·변형

- (여정 5, 1) 익숙한 곡에서 출발하는 건 **현실적인 방법**처럼 느껴진다
- (여정 5, 3) 원곡·연주를 **들으면** 그 진행 **느낌은 알겠다**

### 여정 6 — AI 챗봇·생성 AI

- (여정 6, 1) **"요즘은 AI로 되겠다"** — 새 대안에 대한 기대
- (여정 6, 2) **말로 요청하니까 쉽다** — "밝은 코드 진행 8마디" 등 자연어 입력

### 여정 7 — 포기·미루기

- (여정 7, 1) 밴드·취미 맥락에서 **"작곡하고 싶다"** — 욕구 자체는 유지된다
