# Typography

## Font Families

- Body: Pretendard Variable 기반 산세리프
- Display: Archivo Black (헤드라인/브랜드 포인트용)

로딩 위치: [`web/index.html`](../web/index.html)

토큰 정의: [`web/src/styles/tokens.css`](../web/src/styles/tokens.css)

## Type Scale

- `--text-xs`: 0.75rem
- `--text-sm`: 0.875rem
- `--text-md`: 1rem
- `--text-lg`: 1.125rem
- `--text-xl`: 1.375rem
- `--text-2xl`: 1.75rem

## Weight

- Body 기본: `--weight-body` (400)
- 강조/버튼: 600~700
- Display: `--weight-display` (700)

## Usage Rules

- 일반 UI 라벨/본문은 body 폰트 사용
- `vibe-chord` 로고, 온보딩 타이틀은 display 폰트 사용
- display 폰트 남용 금지: 한 화면에서 1~2곳만 포인트로 적용
